# Required programs to run project
* Visual Studio Code or other source-code editor
* REACT
* Node.js version 16 or greater

# Client side
## Install libraries

$ npm i

## Start project

$ npm start

# Server side
## Declare environment variables
* copy '.env-example' and rename it to '.env'
* fill variables in '.env' file

## Install libraries
$ npm i

## Launch server
$ npm start
